- 15帧

### 技能2 (skill2)

**技能2分为三个阶段：**

#### **skill2_1 (第一阶段)**

- **双手武器**: 使用 picArray[14]
- 8帧
- **特效触发**: 第0帧触发 `addLight("ls5_1")` (特殊技能5-1特效)

- 1000帧循环
- **特点**: 在空中停留，等待落地

#### **skill2_3 (第三阶段)**
- **单手武器**: 使用 picArray[17]
- **双手武器**: 使用 picArray[15]
- 20帧

- **攻击音效**: 第0帧播放 `addSound("skill4")` (技能4音效)
- **特点**: 落地时释放强力攻击，造成屏幕震动

### 翻滚攻击 (rollattack)

**单手武器：**
- 使用 (0)picArray[16] 和 (2)picArray[34]
- 28帧
- **特效触发**: 第2帧触发 `addLight("ls2")` (特殊技能2特效)

- **受击特效**: 命中时显示 `"lh2"` (受击特效2)
- **特点**: 攻击过程中会翻转角色朝向

**双手武器：**
- 使用 (0)picArray[14] 和 (2)picArray[38]
- 28帧
- **特效和音效**: 与单手武器相同
- **特点**: 双手武器翻滚攻击的判定范围更大

### 死亡 (dead)

**单手武器和双手武器：**
- 使用统一的死亡动画
- 20帧
- **特效触发**: 第0帧触发 `addLight("deadMc")` (死亡特效)
- **特点**: 第0帧隐藏角色，第20帧删除角色对象
